# Roadmap

Planned features and enhancements:

- [ ] **Frontend: Edit Task** - Allow editing existing todo items
- [ ] **Frontend: Due Date** - Add due date field to todos with date picker
- [ ] **Frontend: Calendar View** - Visual calendar showing todos by due date
- [ ] **Frontend: Filter View** - Filter todos by status, due date, priority, etc.
- [ ] **Backend: Daily Task Limit** - Configurable maximum tasks per day (default: 10)
- [ ] **Backend: API Integration** - Add backend API (language of choice) with configurable repository pattern
